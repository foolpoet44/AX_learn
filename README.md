# HR AX 실습 교재 (Lovable)

인간은 눈앞에서 변화가 일어날 때 가장 강력하게 배웁니다. 이 프로젝트는 코딩을 전혀 모르는 HR 및 교육 담당자들이 웹 브라우저 위에서 프롬프트 작성부터 앱 동작까지, 모든 과정을 직관적으로 경험할 수 있도록 돕는 인터랙티브 실습 교재입니다. 현재 최신 버전을 기준으로 로직과 에셋이 안정적으로 분리된 형태를 취하고 있습니다.

## 📦 구성 및 아키텍처
단일 페이지 애플리케이션(SPA)의 철학을 담으면서도, 유지보수를 극대화하기 위해 코드를 분리했습니다.
- `index.html`: 메인 뼈대를 구성하는 뷰(View)
- `assets/styles.css`: 디자인 요소와 반응형 그리드를 책임지는 모듈
- `assets/app.js`: 네비게이션, 로컬 스토리지 연동, 인터랙션 및 상태를 관리하는 엔진

## ✨ 핵심 기능과 5가지 실습 과제
각 과제는 '기획 → 프롬프트 → 수정 → 배포의 4단계 여정으로 구성되어 있으며, 사용자의 진행 상황은 자동으로 기억됩니다 (`localStorage`).

1. **1on1 미팅 어젠다 생성기:** 개인화된 면담 계획을 즉각적으로 도출.
2. **역할별 AI 활용 브레인스토머:** 특정 직무를 입력받아 AI 도입 アイ디어를 도출.
3. **JD (Job Description) 생성기:** 주요 요구 역량을 기반으로 직무 기술서를 구조화.
4. **채용 공고 자동 작성기:** 위 JD를 바탕으로 매력적인 공고 포맷팅.
5. **신입사원 온보딩 가이드 앱:** 입사 후 첫 주 활동 및 사내 유용한 링크 안내.

## 🚀 사용 방법 및 실행
설치 과정의 번거로움 없이 본질에만 집중할 수 있습니다.
1. 저장소를 클론하거나 리포지토리 파일들을 다운로드합니다.
2. 폴더 내의 `index.html` 파일을 크롬 등 모던 웹 브라우저로 엽니다.
3. 화면 왼쪽의 내비게이션 바에서 과제를 선택하고 가이드라인에 따라 단계별로 버튼을 클릭하며 진행합니다.

## 🛠 수정 및 커스터마이징 가이드
코드 속에 당신의 문맥을 불어넣고 싶다면 다음 부분을 참고하세요.
- **문구나 과제 내용 변경:** `index.html` 의 해당 텍스트 영역을 직접 수정.
- **테마 컬러 및 디자인 개선:** `assets/styles.css` 내 최상단의 `:root` CSS 변수 조작.
- **비즈니스 로직 및 데모 결괏값 변경:** `assets/app.js` 에 하드코딩된 더미 데이터 및 함수(예: `generateAgenda`, `showRoleIdeas`) 수정.

### 진행 상태 초기화 방법
브라우저의 `개발자 도구 (F12) > Application 탭 > Local Storage` 에서 `hrax_progress` 키를 삭제한 뒤 새로고침하면 모든 진도가 처음 상태로 되돌아갑니다.

## 🌐 Vercel 기반 배포 가이드
작업물 확인을 위해 정적 파일 구조 그 자체로 Vercel에 즉시 배포할 수 있도록 `vercel.json` 룰이 내장되어 있습니다.
```bash
npm install -g vercel
vercel
vercel --prod
```
명령어 몇 번이면 팀원들과 교재를 라이브 링크로 바로 공유할 수 있습니다.
